WinDimmer64 Changelog
====================

v1.0.0 (26 May 2026)
------------------------

[NEW FEATURES]
- [Smooth Fading Transitions] Dimmer overlays now fade in and out gracefully using a high-performance exponential-decay animation system when the application launches, exits, or settings are toggled.
- [Eye-Saver Warm Amber Tint] Introduced an Amber Blue-Light Filter option that transforms standard neutral dimming overlays into curated soothing warm amber light, perfect for night work.
- [System-wide Keyboard Hotkeys] Users can now control active screen brightness globally from any application or game:
  * Ctrl + Alt + ArrowUp   : Brighter (reduces dimming by 5%)
  * Ctrl + Alt + ArrowDown : Darker (increases dimming by 5%)
  * Ctrl + Alt + D         : Toggle all active screen dimmers
- [Focused Screen Highlight (Focus Mode)] Multi-monitor eye helper that tracks the cursor position and smoothly highlights the screen you are currently working on while fading/dimming inactive monitors deeply to reduce eyes distraction.
- [Inactivity Timeout OLED Protection & Energy Saver] Detects system inactivity via GetLastInputInfo. When idle, screens fade smoothly to a user-selectable Dim Level (0% to 100%, perfect for OLED burn-in prevention and power saving on external displays). Moves mouse or presses key to instantly wake monitors up, bypassing slow hardware sleep cycles. Features an option to physically turn off monitors as well.

[IMPROVEMENTS]
- Built exclusively in clean, modern C++ (C++17) targeting Windows 10 & 11 (x64) with native hardware-accelerated Direct2D and DirectWrite.
- Crisp rendering with full Per-Monitor DPI Awareness v2.
- Clean system tray integration with settings panel toggling.
- Portable, lightweight single-executable (< 1MB) with zero external runtime dependencies.
- Re-architected settings parser for ultra-fast performance.
